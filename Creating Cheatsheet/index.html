HTML Cheatsheet

Structure

    This is the basic template or barebone structure of HTML.

  1. HTML Boilerplate 
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>

</body>
</html>

    2. Headings

    There are six heading tag available in HTML from H1 to H6, H1 is the largest among all, and H6 is the smallest

  2. <h1> Tag 
    <h1>Heading 1</h1>
  3. <h2> Tag 
    <h2>Heading 2</h2>
  4. <h3> Tag 
    <h3>Heading 3</h3>
  5. <h4> Tag 
    <h4>Heading 4</h4>
  6. <h5> Tag 
    <h5>Heading 5</h5>
  7. <h6> Tag 
    <h6>Heading 6</h6>

    8. Container

    Container tags are the tags that contain some data such as text, image, etc. There are several container tags in HTML.

  8. div tag

    div tag or division tag is used to make blocks or divisions in the document.

    <div> This is div block </div>
  9. span Tag

    span is a container for inline content

    <span> This is span block </span>
  10. p Tag

    Paragraph

    <p> This is a paragraph </p>
  11. pre Tag

    pre tag represents pre-formatted text

    <pre> Hello World </pre>
  12. code Tag

    Paragraph

    <code> This is div block </code>

    13. Text Formating Tag

    Text formatting tags are used to format text or data of HTML documents. You can do certain things like creating italic, bold, strong text to make your document look more attractive and understandable.

  13. Bold Tag 
    <b> bold text </b>
  14. Strong Tag 
    <strong> strong text </strong>
  15. Italic Tag 
    <i> italic text </i>
  16. Emphasize Tag 
    <em> emphasize text </em>
  17. Subscript Tag 
    <sub> emphasize text </sub>
  18. Superscript Tag 
    <sup> Superscript </sup>

    19. Lists

    Lists can be either numerical, alphabetic, bullet, or other symbols. You can specify list type and list items in HTML for the clean document.

  19. <ol> Tag

    Ordered list starts with <ol> tag and each list item starts with <li> tag

    <ol>
    <li>Data 1</li>
    <li>Data 2</li>
    <li>Data 3</li>
</ol>
  20. <ul> Tag 
    <ul>
    <li>Data 1</li>
    <li>Data 2</li>
    <li>Data 3</li>
</ul>

    21. Media

    Media is anything that is present in digital form such as image, video, audio, etc.

  21. <audio> Tag

    It is used to embed sound content in the document.

    <audio controls>
    <source src="demo.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>
  22. <img> Tag

    It is used to embed or import image in a webpage.

    <img src="Source_of_image" alt="Alternate text">
  23. <audio> Tag 
    <video width="480" height="320" controls>
    <source src="demo_move.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

    24. Table

    A table is a collection of rows and columns. It is used to represent data in tabular form. tr is table row and td table head

  24. Table Structure 
    <table>
    <caption>Demo Table</caption>
    <thead>
        <tr>
            <th>Column1</th>
            <th colspan="2">Column2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data1</td>
            <td>Data2</td>
            <td>Data2</td>
        </tr>
        <tr>
            <td>Data1</td>
            <td>Data2</td>
            <td>Data2</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td> </td>
            <td>Data</td>
            <td>Data</td>
        </tr>
    </tfoot>
</table>

    25. Links

    Links are clickable text that can redirect you to some other page.

  25. <a> tag

    <a> or anchor tag defines a hyperlink.

    <a href="https://www.google.com/">Visit google.com!</a>

    27. Semantic Elements

    Semantic elements are those elements that are self describable, i.e., from their name itself, you can understand their meaning

  27. <section> tag

    It defines a section in the document

    <section>This is a section</section>
  28. <article> tag

    It represents self-contained content

    <article>This is a article</article>
  29. <aside> tag

    It is used to place content in the sidebar

    <aside>This is a aside</aside>
